Inline Visualizer

Hệ thống render SVG/HTML trực tiếp trong chat — chi tiết đủ để tự triển khai lại

1. Tổng quan kiến trúc

Visualizer là một cơ chế nhúng widget inline vào luồng chat. Mỗi widget là một iframe sandbox chứa SVG hoặc HTML fragment, render ngay trong khung chat mà không redirect ra ngoài.

Về bản chất, toàn bộ hệ thống chỉ gồm 3 thành phần:

Thành phầnVai tròCông nghệ
Tool callLLM gọi tool để emit widget codeFunction calling API
RendererHost nhận code, nhúng vào iframeiframe sandbox + postMessage
Widget codeNội dung thực tế được renderSVG hoặc HTML + JS + CSS
Không có thư viện đặc biệt nào. Đây thuần túy là SVG spec + HTML spec + CSS spec. Bạn tự implement host là được.

2. Luồng hoạt động chi tiết

2.1 Phát hiện mode

Host đọc ký tự đầu tiên của widget_code để phân nhánh:

if widget_code.trimStart().startsWith("<svg") → SVG mode
else → HTML mode

2.2 Wrapping vào iframe

Host tạo một full HTML document, nhúng widget code vào, rồi đặt vào iframe srcdoc:

// SVG mode
const doc = `
<!DOCTYPE html>
<html>
<head>
  <style>
    /* inject CSS variables + pre-built classes */
    ${HOST_CSS_VARIABLES}
    ${PREBUILT_CLASSES}
    body { margin: 0; background: transparent; }
    svg { display: block; width: 100%; }
  </style>
</head>
<body>${widget_code}</body>
</html>
`

// HTML mode — widget_code là fragment, không có <html>/<head>/<body>
const doc = `
<!DOCTYPE html>
<html>
<head>
  <style>
    ${HOST_CSS_VARIABLES}
    ${PREBUILT_CLASSES}
    body { margin: 0; padding: 16px; background: transparent; }
  </style>
</head>
<body>
  ${widget_code}
  <script>
    /* inject sendPrompt + openLink bridge */
    window.sendPrompt = function(text) {
      window.parent.postMessage({ type: 'sendPrompt', text }, '*');
    };
    window.openLink = function(url) {
      window.parent.postMessage({ type: 'openLink', url }, '*');
    };
  </script>
</body>
</html>
`

// Nhúng vào iframe
const iframe = document.createElement('iframe');
iframe.srcdoc = doc;
iframe.sandbox = "allow-scripts allow-same-origin";
iframe.style.cssText = "width:100%; border:none; overflow:hidden;";
container.appendChild(iframe);

2.3 Auto-resize iframe

Iframe cần tự báo chiều cao cho host vì nội dung SVG có thể cao bất kỳ:

// Trong iframe — inject vào mỗi document
const observer = new ResizeObserver(() => {
  const h = document.body.scrollHeight;
  window.parent.postMessage({ type: 'resize', height: h }, '*');
});
observer.observe(document.body);

// Ở host — lắng nghe
window.addEventListener('message', (e) => {
  if (e.data.type === 'resize') {
    iframe.style.height = e.data.height + 'px';
  }
  if (e.data.type === 'sendPrompt') {
    chatInput.value = e.data.text;
    chatInput.dispatchEvent(new Event('input'));
    submitButton.click();
  }
});

4. Pre-built CSS Classes

Host cũng inject sẵn các class dùng chung để SVG/HTML code gọn hơn:

4.1 SVG text classes

/* class="t"  → 14px regular */
svg .t  { font-family: var(--font-sans); font-size: 14px; font-weight: 400;
          fill: var(--color-text-primary); }

/* class="ts" → 12px secondary (subtitles, labels) */
svg .ts { font-family: var(--font-sans); font-size: 12px; font-weight: 400;
          fill: var(--color-text-secondary); }

/* class="th" → 14px medium/bold (headings) */
svg .th { font-family: var(--font-sans); font-size: 14px; font-weight: 500;
          fill: var(--color-text-primary); }

4.2 SVG shape classes

/* class="box" → neutral rounded rect */
svg .box { fill: var(--color-background-secondary);
           stroke: var(--color-border-tertiary); }

/* class="arr" → arrow line */
svg .arr { stroke: var(--color-border-secondary);
           stroke-width: 1.5; fill: none; }

/* class="leader" → dashed connector */
svg .leader { stroke: var(--color-text-tertiary);
              stroke-width: 0.5; stroke-dasharray: 3 3; fill: none; }

/* class="node" → clickable group */
svg .node { cursor: pointer; }
svg .node:hover { opacity: 0.82; }

4.3 Color ramp classes (c-*)

Đây là phần quan trọng nhất — 9 ramp màu, mỗi ramp có 7 stops. Class c-{ramp} áp lên <g> hoặc shape element (rect, circle), tự động set fill+stroke+text color cho cả light và dark mode:

/* Định nghĩa ramp (ví dụ c-purple) */
.c-purple rect, .c-purple circle, .c-purple ellipse {
  fill:   #EEEDFE;  /* stop 50 — light mode */
  stroke: #534AB7;  /* stop 600 */
}
.c-purple .th { fill: #3C3489; }  /* stop 800 */
.c-purple .ts { fill: #534AB7; }  /* stop 600 */

@media (prefers-color-scheme: dark) {
  .c-purple rect, .c-purple circle, .c-purple ellipse {
    fill:   #3C3489;  /* stop 800 */
    stroke: #AFA9EC;  /* stop 200 */
  }
  .c-purple .th { fill: #CECBF6; }  /* stop 100 */
  .c-purple .ts { fill: #AFA9EC; }  /* stop 200 */
}

5. SVG Coordinate System

5.1 ViewBox cố định

<svg width="100%" viewBox="0 0 680 H">

680 là bắt buộc — không được thay đổi. Container rộng 680px, width="100%" + viewBox="0 0 680 H" đảm bảo tỷ lệ 1:1 CSS pixel. Nếu dùng viewBox nhỏ hơn (ví dụ 480), toàn bộ content bị scale up 1.42×, text 14px sẽ render thành 20px.

Safe area: x: 40–640, y: 40 đến (H-40).

5.2 Tính chiều cao H

H = max(y + height của tất cả rect) + 40px buffer

5.3 Công thức layout box

/* Box 1 dòng: height = 44px */
Box 1: y=20,  h=44  → bottom = 64
Arrow: y1=64, y2=96 (gap = 32px)
Box 2: y=96,  h=44  → bottom = 140
Arrow: y1=140, y2=172
Box 3: y=172, ...

/* Box 2 dòng: height = 56px */
Title:    y_box + 20  (text-anchor middle, dominant-baseline central)
Subtitle: y_box + 38

5.4 Text trong box — quy tắc bắt buộc

<!-- Luôn dùng text-anchor + dominant-baseline -->
<text class="th"
      x="{x_box + width/2}"
      y="{y_box + height/2}"
      text-anchor="middle"
      dominant-baseline="central">Label</text>

<!-- Không có dominant-baseline → text bị lệch lên ~4px -->

5.5 Ước tính độ rộng text

Ký tự14px weight 50012px weight 400
Latin thường (a-z)~8px/char~7px/char
Uppercase (A-Z)~10px/char~8.5px/char
CJK/Unicode rộng~15px/char~13px/char
Ký tự đặc biệt (₁₂₃)~7px/char~6px/char
rect_width = text_chars × 8 + 24px padding (cả hai bên)

6. Arrow Marker — cấu trúc chuẩn

Phải có trong <defs> của mọi SVG:

<defs>
  <marker id="arrow"
          viewBox="0 0 10 10"
          refX="8" refY="5"
          markerWidth="6" markerHeight="6"
          orient="auto-start-reverse">
    <path d="M2 1L8 5L2 9"
          fill="none"
          stroke="context-stroke"   <!-- kế thừa màu line -->
          stroke-width="1.5"
          stroke-linecap="round"
          stroke-linejoin="round"/>
  </marker>
</defs>

<!-- Dùng -->
<line x1="340" y1="72" x2="340" y2="106"
      stroke="#888" stroke-width="1.5"
      marker-end="url(#arrow)"/>
context-stroke là SVG keyword — mũi tên luôn cùng màu với đường line. Không cần define nhiều marker cho nhiều màu.

8. Interactive HTML Widget

Khi cần slider, animation, JS logic — dùng HTML fragment thay vì SVG thuần. SVG vẫn nằm bên trong HTML:

<!-- HTML fragment — không có DOCTYPE, html, head, body -->
<svg width="100%" viewBox="0 0 680 300">
  <defs>...arrow marker...</defs>

  <rect id="hot-zone" x="180" y="55" width="260" height="120" rx="8"
        fill="#E8593C" opacity="0.3"/>
  <rect x="180" y="55" width="260" height="245" rx="14"
        fill="none" stroke="var(--color-border-secondary)" stroke-width="1"/>
</svg>

<!-- Controls bên dưới SVG -->
<div style="display:flex; gap:16px; align-items:center;
             padding:12px 0; font-size:13px;
             color:var(--color-text-secondary)">
  <span>Nhiệt độ</span>
  <input type="range" min="10" max="90" value="40" id="slider"
         oninput="update(this.value)"
         style="flex:1">
  <span id="label">40%</span>
</div>

<script>
function update(val) {
  document.getElementById('hot-zone').setAttribute('height', val * 2.45);
  document.getElementById('label').textContent = val + '%';
}
</script>

9. sendPrompt() — bridge về chat

Host inject hàm này vào mỗi iframe. Widget gọi để gửi message vào chat:

// Trong widget SVG
<g onclick="sendPrompt('Giải thích MaxPool chi tiết hơn')">
  ...
</g>

// Trong widget HTML
<button onclick="sendPrompt('Backprop qua Conv layer thế nào?')">
  Tìm hiểu thêm
</button>

Phía host implement bridge qua postMessage:

// Inject vào mỗi iframe document
window.sendPrompt = function(text) {
  window.parent.postMessage({ type: 'sendPrompt', text }, '*');
};

// Host lắng nghe
window.addEventListener('message', (event) => {
  if (event.data?.type === 'sendPrompt') {
    // Điền vào chat input và submit
    setInputValue(event.data.text);
    submitChat();
  }
});

11. Iframe Sandbox Settings

<iframe
  srcdoc="${escapedHtml}"
  sandbox="allow-scripts allow-same-origin"
  style="width:100%; border:none; overflow:hidden; display:block;"
  scrolling="no"
></iframe>
Không dùng allow-top-navigation hay allow-forms — không cần thiết và tăng attack surface. allow-same-origin cần để postMessage hoạt động.

13. Checklist trước khi render

  1. ViewBox height = bottom-most element + 40px? ✓
  2. Mọi text đều có class (.t / .ts / .th)? ✓
  3. Mọi connector path đều có fill="none"? ✓
  4. Không có element nào x < 0 hoặc x + width > 680? ✓
  5. Arrow không đi xuyên qua box không liên quan? ✓
  6. Màu chỉ dùng CSS variables hoặc c-* ramp classes? ✓
  7. Tất cả màu đều đọc được ở dark mode? ✓
  8. <defs><marker id="arrow"...> đã có chưa? ✓

14. Ví dụ: Tự build Visualizer host

Đây là React component tối giản để host một widget:

import { useEffect, useRef } from 'react';

function WidgetHost({ widgetCode, onSendPrompt }) {
  const iframeRef = useRef(null);

  // Lắng nghe message từ iframe
  useEffect(() => {
    function handler(e) {
      if (e.data?.type === 'resize') {
        iframeRef.current.style.height = e.data.height + 'px';
      }
      if (e.data?.type === 'sendPrompt') {
        onSendPrompt?.(e.data.text);
      }
    }
    window.addEventListener('message', handler);
    return () => window.removeEventListener('message', handler);
  }, []);

  const isSvg = widgetCode.trimStart().startsWith('<svg');

  const doc = `<!DOCTYPE html><html><head>
    <style>
      ${CSS_VARIABLES}
      ${PREBUILT_CLASSES}
      body { margin: 0; ${isSvg ? '' : 'padding: 16px;'} background: transparent; }
      svg { display: block; width: 100%; }
    </style>
    </head><body>
    ${widgetCode}
    <script>
      window.sendPrompt = t => parent.postMessage({type:'sendPrompt',text:t},'*');
      new ResizeObserver(() =>
        parent.postMessage({type:'resize',height:document.body.scrollHeight},'*')
      ).observe(document.body);
    </script>
    </body></html>`;

  return (
    <div style={{ borderRadius: 12, overflow: 'hidden',
                   border: '1px solid #e0ddd6', margin: '16px 0' }}>
      <iframe
        ref={iframeRef}
        srcDoc={doc}
        sandbox="allow-scripts allow-same-origin"
        style={{ width: '100%', border: 'none',
                 overflow: 'hidden', display: 'block', minHeight: 80 }}
        scrolling="no"
      />
    </div>
  );
}

Tài liệu này đủ để tự build lại Visualizer hoàn chỉnh với React, Vue, hoặc Vanilla JS. Stack duy nhất cần: SVG spec + iframe sandbox + postMessage API.
